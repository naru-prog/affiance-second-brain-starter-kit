## Part 1: VS Code を Mac にインストール（5分）

### Step 1-1: 公式サイトを開く

Safari（または Chrome）で以下にアクセスしてください：

**https://code.visualstudio.com/**

### Step 1-2: ダウンロードボタンをクリック

ページ中央に **大きな青いボタン** があります。Macからアクセスしていれば、自動で「Mac版」と表示されているはずです。

```
   ┌─────────────────────────────────┐
   │   ⬇ Download Mac Universal     │  ← この青ボタンをクリック
   └─────────────────────────────────┘
```

> 💡 「Stable」と「Insiders」の2種類が出ていたら、**必ず「Stable」**（安定版）を選んでください。

### Step 1-3: ダウンロードを待つ

画面右上のSafariのダウンロードマーク（↓）に **`VSCode-darwin-universal.zip`** が表示されます。サイズは約200MB、ネット環境にもよりますが**30秒〜2分**でダウンロード完了します。

### Step 1-4: zip を解凍する

ダウンロードした zip ファイルを **ダブルクリック** してください。
自動で解凍されて、青いアイコンの **「Visual Studio Code」** というアプリが現れます。

```
   ╔═══════════════════╗
   ║   ┌─┐             ║
   ║   │<>│            ║   ← 青いリボン状のアイコン
   ║   └─┘             ║
   ║ Visual Studio Code║
   ╚═══════════════════╝
```

### Step 1-5: アプリケーションフォルダに移動

このアイコンを **「アプリケーション」フォルダ** にドラッグ＆ドロップします。

1. Finder を開く（Dock の一番左のアイコン）
2. サイドバーから **「アプリケーション」** をクリック
3. 先ほど解凍された **「Visual Studio Code」** のアイコンを、**「アプリケーション」のウィンドウ内にドラッグ**

> ⚠️ ここで「アプリケーション」フォルダに入れないと、後でアップデートが正しく動きません。**必ずこの手順を踏んでください。**

### Step 1-6: 初回起動

1. **Launchpad** を開く（F4キー、または Dock のロケット型アイコン）
2. **「Visual Studio Code」** をクリック
3. 「インターネットからダウンロードされたアプリです。本当に開いてもよろしいですか？」と聞かれたら **「開く」** をクリック

VS Code が起動して、暗い画面と「Welcome」タブが表示されたら成功です 🎉

**✅ ここまでできたら Part 2 へ**

## Part 2: VS Code を日本語化する（2分）

VS Code はデフォルトで英語表示です。日本語化しましょう。

### Step 2-1: 拡張機能パネルを開く

VS Code の左サイドバーに5つくらいアイコンが並んでいます。
**上から4番目の「四角が4つ」のアイコン**をクリックしてください。

```
   ┌─┐
   │📁│  ファイル
   ├─┤
   │🔍│  検索
   ├─┤
   │🌿│  Git
   ├─┤
   │⊞ │  ← これ！拡張機能（Extensions）
   ├─┤
   │🐛│  デバッグ
   └─┘
```

### Step 2-2: 「Japanese」を検索

上部の検索ボックスに **`japanese`** と入力してください。

### Step 2-3: 「Japanese Language Pack for Visual Studio Code」をインストール

検索結果の一番上に出る、**Microsoft 公式**のものを選んで **「Install」** ボタンをクリック。

```
   ┌──────────────────────────────────────────────┐
   │ 🇯🇵 Japanese Language Pack for VS Code      │
   │     Microsoft                                 │
   │     [ Install ]   ← クリック                  │
   └──────────────────────────────────────────────┘
```

### Step 2-4: VS Code を再起動

インストール完了後、画面右下に **「Restart」** ボタンが出ます。クリックすると VS Code が再起動して、**全部日本語**になります。

**✅ ここまでできたら Part 3 へ**

## Part 3: Claude Code を VS Code に接続（5分）

ここがこのガイドの**核心**です。AI相棒のClaudeが、VS Code の中で動くようになります。

### Step 3-1: Claude Code 拡張機能を入れる

1. もう一度、左サイドバーの **拡張機能アイコン** をクリック
2. 検索ボックスに **`claude code`** と入力
3. **Anthropic 公式**の **「Claude Code」** を選んで **「インストール」** をクリック

```
   ┌──────────────────────────────────────────────┐
   │ ✨ Claude Code                               │
   │    Anthropic                                  │
   │    [ インストール ]   ← クリック              │
   └──────────────────────────────────────────────┘
```

### Step 3-2: Claude のサインインを開始

インストールが完了すると、左サイドバーに **新しいアイコン（✨星マーク）** が追加されます。それをクリック。

右側のパネルに **「Sign in to Claude」** ボタンが表示されるので、クリックしてください。

### Step 3-3: ブラウザでサインイン

Safari（または既定のブラウザ）が自動で開いて、Anthropicのログインページが表示されます。

| 状況 | やること |
|------|---------|
| すでに Claude.ai のアカウントを持っている | メールアドレスを入力してログイン |
| まだアカウントを持っていない | 「Sign up」をクリックして無料で作成 |

### Step 3-4: VS Code に戻る

ブラウザでログインが完了すると、**「Open Visual Studio Code」** というポップアップが出ます。**「開く」** をクリック。

VS Code に戻ると、Claudeパネルが **使える状態** になっています。

### Step 3-5: 動作テスト

Claude パネルの一番下にあるテキスト入力欄に、以下を打ち込んで Enter:

```
こんにちは。準備はできましたか？
```

Claude が返事をしてくれたら、**接続完了です** 🎉

**✅ ここまでできたら Part 4 へ**

## Part 4: 推奨拡張機能をまとめて入れる（3分）

VS Code 単体でも使えますが、以下の拡張機能を入れると **格段に便利** になります。
ナルさん（このガイドの著者）が実際に使っているものをそのままお勧めします。

### 必須の拡張機能（5つ）

| # | 拡張機能名 | 開発元 | 何ができるか |
|---|-----------|--------|------------|
| 1 | **Claude Code** | Anthropic | ✅ Part 3で導入済み |
| 2 | **Japanese Language Pack** | Microsoft | ✅ Part 2で導入済み |
| 3 | **Draw.io Integration** | Henning Dieterichs | `.drawio` ファイル（フローチャート・組織図）を VS Code 内で直接編集できる |
| 4 | **Markdown Live Editor** | Jun Ishii | Markdown ファイルをリアルタイムプレビュー |
| 5 | **PDF Viewer** | tomoki1207 | PDF を VS Code 内で開ける |

### 便利な拡張機能（4つ）

| # | 拡張機能名 | 開発元 | 何ができるか |
|---|-----------|--------|------------|
| 6 | **Markdown KaTeX** | Jeff Tian | Markdown 内で数式（LaTeX）を表示 |
| 7 | **DocxReader** | Shahil Kumar | Word ファイル（.docx）を VS Code 内で開ける |
| 8 | **Python** | Microsoft | Python のコードを書く・実行する |
| 9 | **Pylance** | Microsoft | Python の高機能アシスタント |

### Mac特有 / 高度な拡張機能（4つ・任意）

| # | 拡張機能名 | 開発元 | 何ができるか |
|---|-----------|--------|------------|
| 10 | **Dev Containers** | Microsoft | Docker コンテナで開発したい人向け |
| 11 | **Container Tools** | Microsoft | Docker コンテナの管理 |
| 12 | **GitHub Copilot Chat** | GitHub | Copilot の有料契約がある人向け |
| 13 | **VS Code Speech** | Microsoft | 音声入力でClaude等と対話 |
| 14 | **VS Code Speech (日本語)** | Microsoft | 上記の日本語パック |

### インストール手順

各拡張機能ごとに、以下を繰り返してください：

1. 左サイドバーの **拡張機能アイコン**（⊞）をクリック
2. 上部検索ボックスに **拡張機能名** を入れて検索
3. 出てきた候補の中から **正しい開発元**（上の表の「開発元」列）のものを選択
4. **「インストール」** をクリック

> 💡 **時短ワザ:** 検索ボックスに `@recommended` と入力すると、ワークスペースで推奨されている拡張機能が一覧表示されます（後述の `.vscode/extensions.json` を作っておけば、ワンクリックで全部入ります）。

**✅ 完了です！**

## トラブルシューティング

| 症状 | 原因 | 対処 |
|------|------|------|
| VS Code が「壊れているため開けません」と言われる | Macのセキュリティ機能 | システム設定 → プライバシーとセキュリティ → 下の方の「このまま開く」をクリック |
| 拡張機能アイコンが見つからない | サイドバーが非表示 | `Cmd + B` でサイドバー表示切替 |
| Claude Code がインストールできない | 検索結果が違うものを表示 | 開発元が **Anthropic** であることを必ず確認 |
| Claude にサインインできない | ブラウザがリダイレクトしない | VS Code を再起動してもう一度試す |
| 日本語化されない | 再起動忘れ | VS Code を完全に終了 (`Cmd + Q`) して開き直す |

## 補足: 拡張機能を一括インストールする方法（上級者向け）

VS Code は、プロジェクトごとに「推奨拡張機能リスト」を持てます。
プロジェクトフォルダの中に `.vscode/extensions.json` というファイルを作ると、そのフォルダを開いた時に右下に **「推奨拡張機能をインストールしますか？」** というポップアップが出て、ワンクリックで全部入ります。

このリポジトリには既に [.vscode/extensions.json](.vscode/extensions.json) を用意してあります。
`my-brain` フォルダを VS Code で開くと自動でインストール案内が出るので、そちらに従ってください。
